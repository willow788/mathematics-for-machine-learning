Mathematics From Scratch: Visual & Computational Projects

This repository contains a growing collection of mathematical projects built from first principles, without relying on high-level numerical libraries.
The goal is to demonstrate a deep understanding of the mathematics behind modern AI/ML through linear algebra, multivariate calculus, vector calculus, and differential equations implemented directly in code.

Each project includes:

Hand-written mathematical derivations

Code implementations using only fundamental tools

Visualizations (3D surfaces, contour plots, vector fields, transformations)

Explanations connecting the math to machine-learning intuition

Modern ML frameworks hide most of the underlying math.
